# 📋 Lista de Requerimientos Multimedia - Fase 4

Para completar la experiencia inmersiva de la aplicación, necesitamos los siguientes archivos de audio y video. Por favor, asegúrate de tenerlos listos o generarlos.

## 📁 Estructura de Carpetas Sugerida
La aplicación buscará los archivos en estas rutas específicas dentro de la carpeta `public/audios/`.

### 1. 🌅 Rutina de Mañana (`public/audios/morning/`)
Audios para comenzar el día con energía y foco.
- [ ] `reprogramacion_morning.mp3` - Audio de afirmaciones/reprogramación matutina.
- [ ] `respiracion_guiada.mp3` - Audio de ejercicios de respiración (ej. Wim Hof o similar).
- [ ] `morning_mix.mp3` (Opcional) - Mezcla de ambos si prefieres un solo archivo largo.

### 2. 🌙 Rutina de Noche (`public/audios/night/`)
Audios para desconectar y dormir profundamente.
- [ ] `sleep_induction_1h.mp3` - Audio largo (1 hora) con música relajante + reprogramación subliminal.
- [ ] `meditacion_noche.mp3` - Meditación guiada para antes de dormir.

### 3. 🧠 Modo Focus / Ambiente (`public/audios/focus/`)
Sonidos de fondo para trabajar o estudiar (loops).
- [ ] `rain.mp3` - Sonido de lluvia constante.
- [ ] `forest.mp3` - Sonido de bosque/naturaleza.
- [ ] `white_noise.mp3` - Ruido blanco o café.
- [ ] `river.mp3` - Sonido de río fluyendo.

### 4. 🆘 Chatbot "Emocional" (`public/audios/sos/`)
Recursos específicos que el Chatbot recomendará según el estado de ánimo.
- [ ] `ansiedad_relief.mp3` - Audio guiado para calmar ataques de ansiedad.
- [ ] `stress_buster.mp3` - Ejercicio rápido para bajar el estrés.
- [ ] `motivation_boost.mp3` - Audio enérgico para momentos de desgana.

## 📝 Instrucciones para el Usuario
1.  **Recopila los archivos**: Busca los audios en tu celular o descárgalos.
2.  **Renombra los archivos**: Usa nombres simples y sin espacios (como en la lista).
3.  **Formato**: Preferiblemente **.mp3** para audios y **.mp4** para videos (si los hay).
4.  **Ubicación**:
    *   Crea las carpetas dentro de `Daily Flow Assistant > public > audios > ...`
    *   Copia y pega los archivos en sus respectivas carpetas.

---
**Nota Técnica**:
Si algún archivo es muy pesado (>50MB), avísame para configurar una carga diferida o externa, ya que podría hacer lenta la aplicación.
